プロフィール

金属加工機械の製造会社にて約10年間勤務。
加工機械システム開発に制御ソフト設計として携わり、C#／C++によるWindowsアプリ開発、アプリおよびロボット操作パネルのUI設計、PLC・ラダーを用いた装置システム開発に従事。
製造業向け制御からXR業界への応用を目指すため、2024年9月からVRプロフェッショナルアカデミーを受講。
実践的な学びを深めるとともに、VRフェスやXRハッカソンにも参加してテーマに沿ったXRコンテンツの企画・開発を経験。
現在はPUN2や空間アンカーを活用したMRマルチプレイコンテンツや産業向けトレーニングアプリの開発に従事しています。

制作物

VAcaAudition

概要

最大100人以上が同時接続可能な大規模VRプレゼンテーション空間を構築するプロジェクトです。リアルタイムでのインタラクティブなプレゼンテーションや、多人数での共同視聴体験を実現します。

特徴的な機能

  • 100人以上の同時接続に対応したマルチプレイヤーVR空間
  • Photon Fusionによる安定した専用サーバー型ネットワーク
  • AVPro Videoを使用した高品質な360度動画と通常動画の同期再生
  • Meta Avatarシステムによるリアルなアバター表現
  • Vivoxによるリアルタイム音声通信機能
  • エモートやライティングエフェクトなどのインタラクティブ要素
  • ゲスト用タブレットUIによる直感的な操作インターフェース
  • Meta Quest専用に最適化されたVR体験

使用言語、ツール

Unity 2022.3 C# Photon Fusion MRTK3 Meta XR SDK Meta Avatar SDK AVPro Video Vivox

XR Layout Planner

概要

VRプロフェッショナルアカデミー主催の第16回VRフェスで作成した、工場の機械レイアウトをMR上で検討する営業向けツールです。

特徴的な機能

  • MR空間上に機械モデルを設置
  • PUN2による複数Quest間のオブジェクト同期
  • 空間アンカーで各Quest間の位置合わせ
  • 他ヘッドセット位置にアバターを重ね合わせ表示
  • ユーザー向け自動追従UIの実装
  • VRoid Studioを用いた3Dアバター作成

使用言語、ツール

Unity6 C# Meta XR SDK VRoid Studio

犬になる薬

概要

VRプロフェッショナルアカデミー主催の第16回XRハッカソンで作成した、ユーザーが犬の姿になってNPCに甘やかされることで、精神的な癒やしを得ることができるセラピーアプリです。

特徴的な機能

  • ユーザーが犬の姿に変身し、NPCから癒やしのアクションを受ける体験を実現
  • ヘッドセットとコントローラーにアバターの手足が追従することで、直感的な操作性を提供
  • UIメニューの実装により、スムーズなインターフェース操作をサポート
  • VRoid Studioを活用した3Dアバターおよび独自のアバターモーションの作成
  • ステージの実装とライティング、パーティクルエフェクト、鏡の導入で臨場感のある空間を演出

使用言語、ツール

Unity 2022 Meta XR SDK VRoid Studio

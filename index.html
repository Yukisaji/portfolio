プロフィール

金属加工機械メーカーで10年にわたり、WindowsアプリやロボットUI、PLCラダー制御などの制御システム開発に従事。
仕様検討から実装・量産・客先対応まで全工程を経験し、現場で使いつづけてもらえるシステムを追求してきました。

開発では現場の声を積極的に取り入れ、誤操作防止やトラブル対応時間の短縮など、ユーザー視点でのUI設計・改善を重視した開発を行ってきました。
また、現場や外部業者との信頼関係を大切にし、相手の立場に立った丁寧な調整・提案を心がけてきました。

XR領域では、VRプロフェッショナルアカデミーでVR/MRアプリ開発を体系的に学び、Photonや空間アンカーを用いたマルチプレイアプリや、産業向けトレーニングアプリ、多人数参加型VR教育イベントシステムの開発に取り組んできました。
非熟練ユーザーでも迷わず使えるチュートリアル導線や、直感的なUIを意識し、ユーザー心理に寄り添った体験設計を実践。
さらに、アバターを活用した自然な自己表現や、多人数が快適に参加できる通信同期の最適化に注力し、没入感のある体験を提供できる仕組みを実現しています。

これまでの製造業システムとXRアプリ開発で培った知見を活かし、分かりやすく、記憶に残り、現場で活用され続ける体験を提供できるアプリケーション設計を強みとしています。

制作物

VR教育イベントシステム

概要

参加者がアバターで登場し、リアルタイムに最大100人が参加可能なVRイベントシステムです。
スライドを用いたプレゼンテーションや、2D・360°動画再生、エモートやエフェクト、アンケートが可能です。

特徴的な機能

  • 100人以上の同時接続に対応したマルチプレイヤーVR空間
  • Meta Avatarシステムによるリアルなアバター表現
  • Photon Fusionによるサーバー型ネットワーク
  • AVPro Videoを使用した2D動画および360度動画の同期再生
  • Vivoxによるリアルタイム音声通信機能
  • エモート、エフェクト、アンケート機能

使用言語、ツール

Unity 2022.3 C# Photon Fusion MRTK3 Meta XR SDK Meta Avatars SDK AVPro Video Vivox

XR Layout Planner

概要

VRプロフェッショナルアカデミー主催の第16回VRフェス作品。 工場の機械レイアウトをMR上で共同検討するためのMRマルチプレイアプリです。

特徴的な機能

  • MR空間上に機械モデルを設置
  • PUN2による複数Quest間のオブジェクト同期
  • 空間アンカーで各Quest間の位置合わせ
  • 他ヘッドセット位置にアバターを重ね合わせ表示
  • ユーザー向け自動追従UIの実装
  • VRoid Studioを用いたアバター3Dモデル作成

使用言語、ツール

Unity6 C# Meta XR SDK VRoid Studio

犬になる薬

概要

VRプロフェッショナルアカデミー主催の第16回XRハッカソンで作成した、ユーザーが犬の姿になってNPCに甘やかされることで、精神的な癒やしを得ることができるセラピーアプリです。

特徴的な機能

  • ユーザーが犬の姿に変身し、NPCから癒やしのアクションを受ける体験を実現
  • ヘッドセットとコントローラーにアバターの手足が追従することで、直感的な操作性を提供
  • UIメニューの実装により、スムーズなインターフェース操作をサポート
  • VRoid Studioを活用した3Dアバターおよび独自のアバターモーションの作成
  • ステージの実装とライティング、パーティクルエフェクト、鏡の導入で臨場感のある空間を演出

使用言語、ツール

Unity 2022 Meta XR SDK VRoid Studio
